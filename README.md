# 🎓 Student Exam Score Prediction

A Machine Learning regression project that predicts student exam performance using academic, demographic, and behavioral features.

---

## 📌 Project Overview

This project uses student-related data such as:

- Attendance
- Study hours
- Assignments
- Participation
- Parent education
- Internet access
- Previous grades
- Health
- Activities

to predict a student's final exam score or GPA using regression algorithms.

---

## 📊 Dataset Information

Dataset Name:
Higher Education Students Performance Evaluation

Dataset Source:
https://www.kaggle.com/datasets/csafrit2/higher-education-students-performance-evaluation

### Dataset Features

- 14,000+ rows
- 30+ columns
- Mixed categorical and numerical data
- Suitable for regression and performance analytics

### Target Variable

- Final Exam Score
OR
- GPA / Grade
